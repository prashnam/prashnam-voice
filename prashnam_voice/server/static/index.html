    Quick start

    prashnam-voice is a local English → Indian-language poll TTS. Audio + project state lives in this folder.

    1. Create a project. Pick a type — Poll (1 question + N options) or Announcement (flat body segments).
    2. Type the English text in any segment. After 5 seconds of no typing, prashnam-voice translates and synthesizes audio in every selected language.
    3. Listen and re-roll. Each language cell has its own audio player and ⟳ button. Use the takes dropdown to switch between previous attempts — the model is non-deterministic, so a re-roll often improves a clip.
    4. Pacing & voices live in Project Settings. Per-language overrides win over the project default.
    5. Pronunciation lexicon in Settings fixes proper nouns: BJP=bee jay pee per line. Per-language overrides take precedence.
    6. Templates wrap each segment with the polling preamble and option-numbering. Toggle the preset pill on a segment to opt out for that one.
    7. Bulk import. "Import CSV…" on the project list lets you create dozens of projects from a spreadsheet.

    Where things live

    • ./projects/<id>/project.json — segments, translations, settings, pointers to current takes.
    • ./projects/<id>/audio/<seg>/<lang>/<attempt>.mp3 — every regen attempt is kept; nothing is overwritten.
    • ~/.config/prashnam-voice/config.json — adapter selection (local / Sarvam) + API keys.
    • ~/.cache/prashnam-voice/audio/ — content-addressed audio cache.

    Reset onboarding

    If you want to switch engines later (e.g. local → Sarvam, or vice-versa), open ~/.config/prashnam-voice/config.json and set onboarded to false, then reload — the wizard runs again.

    Import projects from CSV

    Bulk-create polls or announcements from a CSV. The file's group_id column groups rows into projects.

    Type
    CSV schema reference 
    # poll
group_id,type,english
poll1,question,Who will win the election?
poll1,option,Party A
poll1,option,Party B

# announcement
group_id,english
ann1,Welcome to the meeting
ann1,Please remain seated

    Optional columns: name (project name), langs (pipe-separated; e.g. hi|ta|en).
