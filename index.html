1 Install Python 3.11

prashnam-voice is a Python application. Get the official 3.11 installer below — newer Python releases (3.13, 3.14) often lack prebuilt wheels for the ML libraries we rely on, so install fails. 3.11 is rock solid.

Download Python 3.11 →

On the page that opens, scroll to Files and pick: macOS 64-bit universal2 installer, Windows installer (64-bit), or your Linux distro's package.

2 Run install.py

Once Python is installed, the next step is one file: install.py. It creates a virtual environment, installs dependencies, and starts the local server. Re-running it just updates and restarts.

macOS — double-click install.py in Finder. Python Launcher (installed by python.org) opens it in Terminal.

Windows — double-click install.py in Explorer. The Python Launcher (py.exe) runs it in a console window.

Linux (or anywhere via terminal) — open a terminal in this folder and run python3 install.py.

Keep the window open while you use the app — closing it stops the server. First run downloads ~500 MB of dependencies; subsequent runs are fast.

Double-click opened a text editor?

That means your system isn't associating .py with Python. Open a terminal in this folder and run python3 install.py instead. (On macOS, the python.org installer should have set this up; if you used Homebrew or pyenv, the terminal route is the way.)
